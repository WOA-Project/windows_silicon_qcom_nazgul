# Windows on Snapdragon 835 Board Support Package

Welcome to the Snapdragon 835 Windows BSP Platform repository.

This repository contains drivers, components and files needed to run Windows on official Snapdragon 835 platforms such as:

- Mobile Testing Platform (MTP)
- Hardware Development Kit (HDK)

We currently support the following chipsets:

- Snapdragon 835

In the following variants:

- MSM
- APQ
